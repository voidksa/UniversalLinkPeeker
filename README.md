## üéÆ Usage

1. **Launch**: Run the app; it sits quietly in your system tray.
2. **Hover**: Move your mouse over any link.
3. **Peek**: Hold **Shift** (default) to open the preview.
4. **Interact**: 
   - **Scroll**: Use mouse wheel.
   - **Copy**: Press `C`.
5. **Dismiss**: Release the key to close.
